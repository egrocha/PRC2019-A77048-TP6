Resumo:

Este trabalho envolveu a importação de dados sobre Portugal contidos em formato XLSX para a base de dados GraphDB.

Para realizar esse objetivo, foi criado um script em Python para por os dados num ficheiro TTL, que pode então ser importado para o GraphDB.

De modo a facilitar o trabalho do script, os dados XLSX foram primeiro convertidos em CSV por uma ferramenta online.

Depois de termos o output do script, concatenamos as declarações de Classes e Relações do ficheiro TTL usado na aula nº 6 para obter o ficheiro final.
